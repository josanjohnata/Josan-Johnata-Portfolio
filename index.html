The development of this project aimed to identify the churn generation of customers. Pentesting

Pentesting is a way to detect and exploit existing vulnerabilities in systems, that is, to simulate hacker attacks. These assessments are useful for validating the effectiveness of the application's defense mechanisms and the servers behind it.
Testing can be performed manually, but is usually supported by automated tools.
The ultimate purpose is to assess any consequences that security breaches might have on the resources or operations involved.
